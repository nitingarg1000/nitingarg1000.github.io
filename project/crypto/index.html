Breaking Cryptosystems

Apr 27, 2020 misc

This project was a part of a course (CS641: Modern Cryptology) taught by Prof. Manindra Agrawal. We performed Cyptanalysis of Substitution cipher, Block substitution cipher, Substitution-Permutation cipher. We also analyzed special cases of cryptanalysis of DES (Differential cryptanalysis), AES (SASAS attack), RSA with low exponent (Coppersmith attack). Towards the end, we learnt and worked through different attacks on weaker version on KECCAK hashing. We used python for implementing these cryptographic attacks and were able to break each of the above mentioned algorithms.
